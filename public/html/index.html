Thoughts

Learning to Love the Language of the Web

I'm really really starting to like javascript.

Like many other people I have talked to, when I first started learning JS I continuously found myself wondering how anyone uses this language productively. However, after embracing the functional aspects of the language rather than trying to squish it messily into the object oriented paradigms I was used to, my enjoyment of this language has improved dramatically and my code is cleaner too. A big part of this was reading JavaScript: The Good Parts, which has helped me avoid the many common pitfalls of JavaScript such as global variables. It was also helpful in understanding the prototypal inheritance system of JavaScript as opposed to the classical inheritance I was used to from Java.

In the span of just a few weeks JavaScript has gone from something I reluctantly used for front end programming to something I find myself actually wanting to program in. I've since begun to immerse myself in the vibrant JavaScript ecosystem. Tools such as React, which I used on a recent project, have really impressed me with the way they allow you to build rich client-side single page apps out of small, composable components. I've also begun making heavy use of Ramda recently which is an amazing toolkit of functions that brings a functional programming flavor to JavaScript with minimal effort.
